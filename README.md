# 我爱KTV

我爱KTV是一款面向家庭娱乐、包厢点歌和大屏播放场景的 KTV 点歌应用。它将点歌、排队、播放控制和歌库管理整合到同一套界面中，帮助用户用更直接的方式完成找歌、点歌、播放和切歌。

## 产品定位

这是一套以“点歌效率”和“播放连续性”为核心的 KTV 产品：

- 适合连接电视、投屏屏幕或横屏设备使用
- 支持本地歌库与云端歌库统一管理
- 兼顾点歌操作、播放控制和下载管理
- 强调大屏可读性与多人轮流点歌体验

## 核心功能

### 多入口点歌

首页提供清晰的点歌入口，用户可以按不同习惯快速进入歌库：

- 按歌名点歌
- 按歌星点歌
- 浏览本地歌曲
- 查看收藏歌曲
- 查看常唱歌曲

## 智能搜索体验

应用支持围绕点歌场景的快速检索，减少翻页和查找成本：

- 支持按歌名搜索
- 支持按歌手搜索
- 支持中文和拼音首字母检索
- 在本地歌曲场景下可按文件名辅助查找

## 播放与控场

围绕 KTV 使用过程中的高频操作，应用内提供完整的播放控制能力：

- 播放 / 暂停
- 切歌
- 重唱
- 原唱 / 伴唱切换
- 当前播放状态展示
- 大屏预览与全屏播放衔接

这套控制能力适合包厢多人轮流点歌，也适合家庭场景下由一人统一控场。

## 已点队列管理

点歌后，歌曲会进入统一的已点列表，方便随时查看和调整顺序：

- 展示当前播放歌曲
- 查看后续待播队列
- 支持将歌曲置顶
- 支持从队列中移除歌曲
- 支持在已点列表内继续搜索

## 歌库来源

我爱KTV支持多种歌曲来源，便于在不同使用环境下灵活组建歌库：

- 本地目录歌曲
- 百度网盘歌曲源

本地与云端歌曲可以纳入同一套点歌流程中，既适合已有本地曲库的场景，也适合按需从云端补充歌曲资源。

## 下载管理

针对云端歌曲，应用内提供下载能力与任务管理能力：

- 支持将云端歌曲下载到本地
- 可查看未完成下载任务
- 可查看已下载歌曲列表
- 支持恢复下载、删除已下载内容等管理操作

这使得用户既可以在线播放，也可以逐步沉淀自己的本地歌库。

## 个性化歌单能力

为了提高复唱效率，应用内提供贴近 KTV 习惯的沉淀方式：

- 收藏歌曲，方便重复点唱
- 常唱歌曲聚合，快速回到熟悉曲目

对于经常使用同一套歌库的场景，这类能力可以明显缩短找歌路径。

## 设置中心

设置页围绕“歌库配置”展开，帮助用户完成日常使用前的准备工作：

- 配置本地歌曲目录
- 登录并接入百度网盘
- 设定云端歌曲根目录
- 查看和管理下载内容

## 适用场景

- 家庭客厅 KTV
- 小型包厢点歌机
- 本地歌库管理与播放
- 云端歌库补充与下载
- 适合需要原唱 / 伴唱切换的演唱场景

## 典型使用流程

1. 先在设置中接入本地目录或百度网盘歌库
2. 从首页进入歌名、歌星、本地、收藏或常唱入口
3. 通过搜索快速找到目标歌曲并加入已点列表
4. 在播放过程中随时切歌、重唱，或切换原唱与伴唱
5. 对需要保留的云端歌曲执行下载，逐步沉淀为本地资源

## 产品价值

我爱KTV不是单纯的播放器，而是一套围绕“点歌 + 播放 + 管理”完整展开的 KTV 使用体验。它希望把找歌更快、排队更清楚、播放更稳定、歌库更好管理这几件事统一起来，让用户更专注于唱歌本身。
